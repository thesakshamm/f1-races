# f1-races

-this is a f1 race bot whihc sends message in viber channel if there is race tommorrow and for now it is being tested so it also sends 'no race tommorrow message'.
- its being automated thorough github actions --> .github/workflows/.yml file
-api keys are to be kept in .env files and .env files are kept in gitignore
-to load the keys from .env file python-dotenv library is being used
-webhook setup and user id was figure out through thunderClient extenstion in VS CODE
